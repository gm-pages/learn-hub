Métodos de Cálculo · True Sidereal

Gráficos True Sidereal Human Design

A astrologia True Sidereal utiliza as posições visuais reais das constelações no céu. A Genetic Matrix oferece três variantes True Sidereal - Kanatas, Chimenti e Midpoint - cada uma utilizando uma abordagem diferente para mapear os limites das constelações.

| Métodos de Cálculo Disponíveis

True Sidereal é uma família de métodos de cálculo que visam reflectir o que realmente está a acontecer no céu no momento de um cálculo de gráfico. Em vez de usar divisões iguais de signos de 30 graus ou um único valor de desfasamento, os sistemas True Sidereal trabalham com as constelações como realmente aparecem - o que significa que os signos zodiacais podem ter tamanhos diferentes e os limites entre eles dependem do sistema de calibração escolhido.

Isto é importante para o Human Design porque o sistema divide o zodíaco em subdivisões muito refinadas - 64 Portas, 384 Linhas, e camadas mais profundas de Cor, Tom e Base. Nesta resolução, mesmo pequenas diferenças no local onde um limite de constelação se situa podem alterar o que aparece no gráfico.

| Três Sistemas True Sidereal na Genetic Matrix

A diferença principal entre as três opções True Sidereal é como cada uma define o ponto de partida e os limites das constelações. Todas as três utilizam o céu visível como referência em vez do equinócio sazonal, mas discordam sobre exactamente onde uma constelação termina e a seguinte começa.

True Sidereal-K

Kanatas

O ayanamsa Kanatas, desenvolvido pelo astrónomo grego Vasilis Kanatas, utiliza limites de constelações que incorporam Ofiúco como a 13ª constelação zodiacal. O Sol passa diferentes quantidades de tempo em cada signo - desde apenas 8 dias em Escorpião até mais de 44 dias em Virgem. Este sistema é amplamente usado na astrologia sideral ocidental juntamente com o ayanamsa Fagan-Bradley.

True Sidereal-C

Chimenti

O ayanamsa Chimenti, desenvolvido pelo astrólogo Athen Chimenti do Mastering the Zodiac, utiliza a estrela fixa Beta Aries (Sharatan) como ponto de calibração, colocando-a a 2°15' de Carneiro no zodíaco True Sidereal. Este sistema também utiliza os tamanhos reais das constelações e é considerado por muitos profissionais como mais consistente com os limites tradicionais das constelações.

True Sidereal-M

Midpoint

O ayanamsa Midpoint, também criado pelo Mastering the Zodiac, utiliza os tamanhos reais das constelações como o sistema IAU - mas com limites astrologicamente definidos em vez de astronómicos. Os limites da IAU foram concebidos para catalogação de observatórios, não para astrologia, e não fazem referência à eclíptica (o caminho do Sol e dos planetas). O sistema Midpoint corrige isto.

Porquê Três Opções?

Não existe um único sistema True Sidereal "correcto" - cada um representa uma abordagem diferente à mesma questão fundamental: onde exactamente começam e terminam as constelações? Ao oferecer os três, a Genetic Matrix permite comparar resultados e determinar qual sistema produz um gráfico que mais exactamente reflecte a experiência pessoal.

| True Sidereal vs Outros Sistemas

Para compreender o que é o True Sidereal, ajuda ver o que não é. A distinção principal é como cada sistema define o zodíaco - o círculo de 360 graus usado para mapear as posições planetárias.

Tropical - Ancorado às Estações

O zodíaco Tropical define 0° de Carneiro como o ponto onde o Sol cruza o equador celeste no equinócio de março. É uma estrutura sazonal - ligada à relação da Terra com o Sol, não às estrelas. Este é o sistema que Ra Uru Hu usou para o Human Design, e permanece o método de cálculo mais comum. O zodíaco está dividido em doze signos iguais de 30 graus.

Sideral Padrão - Ancorado às Estrelas, Signos Iguais

O Sideral padrão (como o ayanamsa Lahiri usado em grande parte da astrologia indiana) desloca todo o zodíaco por um valor de desfasamento - o ayanamsa - para considerar a precessão dos equinócios. Isto alinha os signos mais proximamente com as estrelas fixas. No entanto, ainda divide o zodíaco em doze signos iguais de 30 graus, que não correspondem aos tamanhos reais das constelações no céu.

True Sidereal - Ancorado ao Céu Visível

O True Sidereal vai mais longe. Em vez de aplicar um simples desfasamento e manter tamanhos iguais de signos, trabalha com as constelações como realmente aparecem. Algumas constelações estendem-se muito mais que 30 graus (Virgem cobre aproximadamente 44 graus), enquanto outras são muito menores (Escorpião abrange apenas cerca de 8 graus). O resultado é um zodíaco desigual que reflecte a realidade astronómica do céu.

Exemplo do Quotidiano

Imagine que está a mapear bairros numa cidade. A abordagem Tropical divide a cidade em doze zonas perfeitamente iguais independentemente de onde os bairros reais se situam. O Sideral padrão desloca essas zonas para melhor coincidir com os bairros, mas ainda as torna todas do mesmo tamanho. O True Sidereal traça os limites reais de cada bairro - alguns grandes, alguns pequenos - como realmente existem no terreno.

| O Que Pode Mudar no Seu Gráfico

Como o True Sidereal altera as posições planetárias e utiliza limites de signos diferentes comparado ao Tropical, as diferenças num gráfico de Human Design podem ser significativas. Ao contrário da comparação Tropical-para-J2000 onde as diferenças tendem a aparecer primeiro a níveis refinados, mudar para True Sidereal pode alterar colocações por 20 graus ou mais - suficiente para mudar quase tudo.

Elementos Que Podem Mudar

A Porta Solar - e com ela, potencialmente a Cruz de Encarnação - pode mudar para uma porta completamente diferente. O Perfil (as Linhas do Sol da Personalidade e Design) pode mudar inteiramente. Os Canais podem aparecer ou desaparecer à medida que as portas planetárias se alteram, o que afecta a definição dos Centros e pode portanto mudar o Tipo e Autoridade.

Nos níveis mais refinados, as Cores, Tons e Bases quase certamente serão diferentes, o que significa que a Variável - a arquitectura cognitiva - pode mostrar uma configuração completamente diferente.

Exemplo do Quotidiano

Se mudar do Tropical para J2000 é como recalibrar o GPS para corrigir um pequeno desvio, mudar para True Sidereal é como usar uma projecção de mapa completamente diferente. O território é o mesmo - os planetas estavam exactamente onde estavam no nascimento - mas o sistema de coordenadas que descreve as suas posições é fundamentalmente diferente.

Importante

Um gráfico diferente não significa que o gráfico anterior estava "errado". Tropical e True Sidereal são sistemas de coordenadas diferentes descrevendo o mesmo céu. Muitas pessoas encontram valor em ambos. A questão é qual estrutura produz um gráfico que descreve mais exactamente a experiência vivida - e só cada pessoa pode responder a isso.

| Precessão e o Ayanamsa

A razão pela qual os zodíacos Tropical e Sideral discordam é um fenómeno chamado precessão dos equinócios. O eixo da Terra oscila lentamente como um pião, traçando um círculo no espaço durante aproximadamente 26.000 anos. Isto significa que o ponto onde o Sol cruza o equador celeste no equinócio da primavera - 0° Carneiro no sistema Tropical - deriva lentamente em relação às estrelas fixas.

Há cerca de 2.000 anos, os zodíacos Tropical e Sideral estavam aproximadamente alinhados. Hoje, a diferença (chamada ayanamsa) é de aproximadamente 24 graus - o que significa que um planeta a 0° Carneiro em Tropical estaria cerca de 6° Peixes em Sideral. Esta deriva continua a aproximadamente 1 grau a cada 72 anos.

O Sideral padrão tem isso em conta aplicando um único desvio. O True Sidereal vai mais longe, ajustando também para o facto de as constelações não serem iguais em tamanho - e depois as três variantes (Kanatas, Chimenti, Midpoint) diferem sobre exatamente como definir as fronteiras das constelações ao longo da eclíptica.

Exemplo do Dia a Dia

A precessão é como um relógio cujas marcações de horas rodaram lentamente em relação aos ponteiros. A hora não mudou - o meio-dia continua a ser meio-dia - mas o número para o qual o ponteiro das horas aponta deslocou-se. O Tropical lê o relógio pela posição atual das marcações. O Sideral lê-o por onde as marcações estavam quando o relógio foi construído. O True Sidereal diz que as marcações nunca estiveram espaçadas de forma uniforme.

| Perguntas Frequentes

O que é a astrologia True Sidereal?
A astrologia True Sidereal utiliza as posições visíveis reais das constelações no céu para definir o zodíaco. Ao contrário do Sideral padrão, que usa signos de 30 graus iguais com um desvio fixo do Tropical, o True Sidereal tem em conta o facto de as constelações serem de tamanhos diferentes. Algumas constelações abrangem mais de 40 graus enquanto outras abrangem menos de 10. O True Sidereal reflete esta realidade astronómica.
O que é um ayanamsa?
Ayanamsa é um termo sânscrito que significa a diferença angular entre os zodíacos Tropical e Sideral. Tem em conta a precessão dos equinócios - a deriva lenta do eixo da Terra que faz com que o zodíaco sazonal e o zodíaco baseado nas estrelas se separem gradualmente. Diferentes ayanamsas (Kanatas, Chimenti, Midpoint, Lahiri, Fagan-Bradley, etc.) usam diferentes estrelas de calibração ou métodos, razão pela qual os gráficos Siderais podem variar dependendo do ayanamsa usado.
Qual é a diferença entre Kanatas, Chimenti e Midpoint?
Os três são sistemas True Sidereal que usam os tamanhos reais das constelações. Diferem na forma como definem as fronteiras das constelações. O Kanatas usa o seu próprio sistema de fronteiras. O Chimenti calibra-se à estrela fixa Beta Carneiro (Sharatan) a 2°15' Carneiro. O Midpoint usa os pontos médios entre onde as constelações cruzam a eclíptica, corrigindo as fronteiras astronómicas da IAU para serem astrologicamente relevantes. As diferenças entre os três são tipicamente pequenas - mas nas subdivisões refinadas do Human Design, mesmo pequenas diferenças podem produzir resultados de gráfico diferentes.
O True Sidereal inclui Ofiúco?
Sim. As três variantes True Sidereal no Genetic Matrix incluem Ofiúco como uma constelação pela qual o Sol passa. No sistema Kanatas, o Sol passa aproximadamente 18 dias em Ofiúco (aproximadamente 7-18 de dezembro). No entanto, como Ofiúco é tratado num contexto de Human Design depende do sistema específico e dos seus mapeamentos de Portas.
O meu Tipo mudará com o True Sidereal?
Pode mudar. Como o True Sidereal pode deslocar posições planetárias em 20+ graus comparado com o Tropical, as Portas e Canais podem ser completamente diferentes. Se isto alterar quais Centros estão definidos no Bodygraph, o Tipo e Autoridade também podem mudar. Algumas pessoas descobrem que o seu gráfico True Sidereal confirma elementos que sempre sentiram ser verdadeiros mas não viam no seu gráfico Tropical. Outras descobrem que o seu gráfico Tropical continua a ser a descrição mais precisa.
O True Sidereal é mais preciso que o Tropical?
Isto depende do que se entende por "preciso." O Tropical tem sido usado com sucesso há milhares de anos na astrologia ocidental e foi o sistema que Ra Uru Hu usou para o Human Design. O True Sidereal reflete as posições astronómicas como aparecem no céu. São estruturas diferentes, cada uma com a sua própria lógica e validade. O Genetic Matrix oferece ambos para que se possa comparar e determinar qual sistema melhor descreve a experiência.
Como é que o True Sidereal é diferente do Sideral padrão?
O Sideral padrão (como o ayanamsa Lahiri) pega no zodíaco Tropical e desloca-o por um desvio fixo para realinhar com as estrelas, mas mantém todos os doze signos com exatamente 30 graus cada. O True Sidereal vai mais longe - usa os tamanhos reais das constelações, que variam significativamente. Isto significa que o True Sidereal pode produzir resultados substancialmente diferentes do Sideral padrão, especialmente para colocações perto das fronteiras dos signos.
Que variante True Sidereal devo usar?
Não há uma resposta universal. Cada variante representa uma abordagem válida para mapear o céu visível. Recomendamos gerar o gráfico com as três e comparar os resultados. Muitos praticantes que trabalham com True Sidereal recomendam o ayanamsa Chimenti pela sua consistência com as referências tradicionais das constelações, enquanto o Kanatas é amplamente usado na astrologia sideral ocidental. O sistema Midpoint oferece uma alternativa astrologicamente refinada às fronteiras astronómicas da IAU.
Ra Uru Hu aprovou o True Sidereal?
Ra Uru Hu desenvolveu o Human Design usando o zodíaco Tropical. Não usou cálculos Siderais ou True Sidereal. O Genetic Matrix oferece o True Sidereal como uma ferramenta de exploração adicional para aqueles que desejam ver o seu gráfico através desta lente. Fornecemos ferramentas, não doutrina - e não afirmamos saber o que Ra teria ou não aprovado.

Esta é uma forma útil de avaliar o sistema - observe gráficos de pessoas cujas vidas e qualidades já conhece, e veja que método de cálculo reflete mais precisamente o que observa sobre elas.
