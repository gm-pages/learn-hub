Gráficos True Sidereal Human Design

A astrologia True Sidereal usa as posições visíveis reais das constelações no céu. A Genetic Matrix oferece três variantes True Sidereal - Kanatas, Chimenti e Midpoint - cada uma usando uma abordagem diferente para mapear os limites das constelações.

True Sidereal é uma família de métodos de cálculo que visa refletir o que está realmente a acontecer no céu no momento do cálculo de um gráfico. Em vez de usar divisões de signos iguais de 30 graus ou um único valor de desvio, os sistemas True Sidereal trabalham com as constelações como realmente aparecem - o que significa que os signos do zodíaco podem ter tamanhos diferentes e os limites entre eles dependem do sistema de calibração escolhido.

Isto importa para o Human Design porque o sistema divide o zodíaco em subdivisões muito finas - 64 Portas, 384 Linhas, e camadas mais profundas de Cor, Tom e Base. Nesta resolução, mesmo pequenas diferenças em onde cai um limite de constelação podem mudar o que aparece no gráfico.

| Três Sistemas True Sidereal na Genetic Matrix

A diferença principal entre as três opções True Sidereal é como cada uma define o ponto de partida e os limites das constelações. As três usam o céu visível como referência em vez do equinócio sazonal, mas discordam sobre exatamente onde uma constelação termina e a seguinte começa.

True Sidereal-K

Kanatas

O ayanamsa Kanatas, desenvolvido pelo astrónomo grego Vasilis Kanatas, usa limites de constelações que incorporam Ofiúco como a 13ª constelação zodiacal. O Sol passa diferentes quantidades de tempo em cada signo - de apenas 8 dias em Escorpião a mais de 44 dias em Virgem. Este sistema é amplamente usado na astrologia sideral ocidental juntamente com o ayanamsa Fagan-Bradley.

True Sidereal-C

Chimenti

O ayanamsa Chimenti, desenvolvido pelo astrólogo Athen Chimenti de Mastering the Zodiac, usa a estrela fixa Beta Carneiro (Sharatan) como ponto de calibração, colocando-a a 2°15' de Carneiro no zodíaco True Sidereal. Este sistema também usa tamanhos reais de constelações e é considerado por muitos praticantes como mais consistente com os limites tradicionais das constelações.

True Sidereal-M

Midpoint

O ayanamsa Midpoint, também criado por Mastering the Zodiac, usa os tamanhos reais das constelações como o sistema IAU - mas com limites definidos astrologicamente em vez de astronómicos. Os limites IAU foram desenhados para catalogação de observatórios, não para astrologia, e não fazem referência à eclíptica (o caminho do Sol e planetas). O sistema Midpoint corrige isto.

Porquê Três Opções?

Não existe um sistema True Sidereal único "correto" - cada um representa uma abordagem diferente à mesma questão fundamental: onde exatamente começam e terminam as constelações? Ao oferecer os três, a Genetic Matrix permite comparar resultados e determinar qual sistema produz um gráfico que mais precisamente reflete a sua experiência.

| True Sidereal vs Outros Sistemas

Para entender o que é True Sidereal, ajuda ver o que não é. A distinção principal é como cada sistema define o zodíaco - o círculo de 360 graus usado para mapear posições planetárias.

Tropical - Ancorado às Estações

O zodíaco Tropical define 0° de Carneiro como o ponto onde o Sol cruza o equador celeste no equinócio de março. É uma estrutura sazonal - ligada à relação da Terra com o Sol, não às estrelas. Este é o sistema que Ra Uru Hu usou para o Human Design, e continua a ser o método de cálculo mais comum. O zodíaco está dividido em doze signos iguais de 30 graus.

Sideral Padrão - Ancorado às Estrelas, Signos Iguais

O Sideral padrão (como o ayanamsa Lahiri usado em muito da astrologia indiana) desloca todo o zodíaco por um valor de desvio - o ayanamsa - para considerar a precessão dos equinócios. Isto alinha os signos mais estreitamente com as estrelas fixas. No entanto, ainda divide o zodíaco em doze signos iguais de 30 graus, que não correspondem aos tamanhos reais das constelações no céu.

True Sidereal - Ancorado ao Céu Visível

True Sidereal vai mais longe. Em vez de aplicar um desvio simples e manter tamanhos de signos iguais, trabalha com as constelações como realmente aparecem. Algumas constelações abrangem muito mais de 30 graus (Virgem cobre aproximadamente 44 graus), enquanto outras são muito menores (Escorpião abrange apenas cerca de 8 graus). O resultado é um zodíaco desigual que reflete a realidade astronómica do céu.

Exemplo Quotidiano

Imagine que está a mapear bairros numa cidade. A abordagem Tropical divide a cidade em doze zonas perfeitamente iguais independentemente de onde os bairros reais se situam. O Sideral padrão desloca essas zonas para melhor corresponder aos bairros, mas ainda as mantém todas do mesmo tamanho. O True Sidereal traça os limites reais de cada bairro - alguns grandes, alguns pequenos - como realmente existem no terreno.

| O Que Pode Mudar no Seu Gráfico

Como o True Sidereal desloca posições planetárias e usa limites de signos diferentes comparado ao Tropical, as diferenças num gráfico de Human Design podem ser significativas. Ao contrário da comparação Tropical-para-J2000 onde as diferenças tendem a aparecer primeiro em níveis de granularidade fina, mudar para True Sidereal pode deslocar posicionamentos em 20 graus ou mais - suficiente para mudar quase tudo.

Elementos Que Podem Mudar

A Porta Solar - e com ela, potencialmente a Cruz de Encarnação - pode deslocar-se para uma porta completamente diferente. O Perfil (as Linhas do Sol da Personalidade e do Design) pode mudar inteiramente. Canais podem aparecer ou desaparecer à medida que as portas planetárias se deslocam, o que afeta a definição do Centro e pode portanto mudar o Tipo e Autoridade.

Nos níveis mais finos, as Cores, Tons e Bases vão quase certamente ser diferentes, o que significa que a Variável - a arquitetura cognitiva - pode mostrar uma configuração completamente diferente.

Exemplo Quotidiano

Se mudar de Tropical para J2000 é como recalibrar o GPS para corrigir um pequeno desvio, mudar para True Sidereal é como usar uma projeção de mapa completamente diferente. O território é o mesmo - os planetas estavam exatamente onde estavam no nascimento - mas o sistema de coordenadas que descreve as suas posições é fundamentalmente diferente.

Importante

Um gráfico diferente não significa que o gráfico anterior estava "errado". Tropical e True Sidereal são sistemas de coordenadas diferentes que descrevem o mesmo céu. Muitas pessoas encontram valor em ambos. A questão é qual estrutura produz um gráfico que descreve mais precisamente a experiência vivida - e só cada pessoa pode responder a isso.

| Precessão e o Ayanamsa

A razão pela qual os zodíacos Tropical e Sideral discordam em primeiro lugar é um fenómeno chamado precessão dos equinócios. O eixo da Terra oscila lentamente como um pião, traçando um círculo no espaço ao longo de aproximadamente 26.000 anos. Isto significa que o ponto onde o Sol cruza o equador celeste no equinócio da primavera - 0° de Carneiro no sistema Tropical - deriva lentamente em relação às estrelas fixas.

Há cerca de 2.000 anos, os zodíacos Tropical e Sideral estavam aproximadamente alinhados. Hoje, a diferença (chamada ayanamsa) é de aproximadamente 24 graus - significando que um planeta a 0° de Carneiro em Tropical estaria em volta dos 6° de Peixes em Sideral. Esta deriva continua a cerca de 1 grau a cada 72 anos.

O Sideral padrão considera isto aplicando um único desvio. O True Sidereal vai mais longe ajustando também ao facto de as constelações não serem iguais em tamanho - e depois as três variantes (Kanatas, Chimenti, Midpoint) diferem sobre exatamente como definir os limites das constelações ao longo da eclíptica.

Exemplo Quotidiano

A precessão é como um relógio cujas marcações horais rodaram lentamente em relação aos ponteiros. A hora não mudou - meio-dia continua a ser meio-dia - mas o número para o qual o ponteiro das horas aponta deslocou-se. Tropical lê o relógio pela posição atual das marcações. Sideral lê-o por onde as marcações estavam quando o relógio foi construído. True Sidereal diz que as marcações nunca estiveram uniformemente espaçadas para começar.

| Perguntas Frequentes

O que é a astrologia True Sidereal?
A astrologia True Sidereal usa as posições visíveis reais das constelações no céu para definir o zodíaco. Ao contrário do Sideral padrão, que usa signos iguais de 30 graus com um desvio fixo do Tropical, o True Sidereal considera o facto de as constelações terem tamanhos diferentes. Algumas constelações abrangem mais de 40 graus, enquanto outras abrangem menos de 10. O True Sidereal reflete esta realidade astronómica.
O que é um ayanamsa?
Ayanamsa é um termo sânscrito que significa a diferença angular entre os zodíacos Tropical e Sideral. Considera a precessão dos equinócios - a deriva lenta do eixo da Terra que faz com que o zodíaco sazonal e o zodíaco baseado nas estrelas se separem gradualmente. Diferentes ayanamsas (Kanatas, Chimenti, Midpoint, Lahiri, Fagan-Bradley, etc.) usam diferentes estrelas de calibração ou métodos, razão pela qual os gráficos siderais podem variar dependendo de qual ayanamsa é usado.
Qual é a diferença entre Kanatas, Chimenti e Midpoint?
Os três são sistemas True Sidereal que usam os tamanhos reais das constelações. Diferem na forma como definem os limites das constelações. Kanatas usa o seu próprio sistema de limites. Chimenti calibra à estrela fixa Beta Carneiro (Sharatan) a 2°15' de Carneiro. Midpoint usa os pontos médios entre onde as constelações cruzam a eclíptica, corrigindo os limites astronómicos da IAU para serem astrologicamente relevantes. As diferenças entre os três são tipicamente pequenas - mas nas subdivisões finas do Human Design, mesmo pequenas diferenças podem produzir resultados de gráfico diferentes.
O True Sidereal inclui Ofiúco?
Sim. Todas as três variantes True Sidereal na Genetic Matrix incluem Ofiúco como uma constelação pela qual o Sol passa. No sistema Kanatas, o Sol passa aproximadamente 18 dias em Ofiúco (aproximadamente de 7 a 18 de dezembro). No entanto, como Ofiúco é tratado num contexto de Human Design depende do sistema específico e dos seus mapeamentos de portas.
O meu Tipo vai mudar com o True Sidereal?
Pode. Como o True Sidereal pode deslocar posições planetárias em mais de 20 graus comparado ao Tropical, portas e canais podem ser completamente diferentes. Se isto mudar quais centros estão definidos no Bodygraph, o Tipo e Autoridade também podem mudar. Algumas pessoas descobrem que o seu gráfico True Sidereal confirma elementos que sempre sentiram ser verdadeiros mas não viram no seu gráfico Tropical. Outras descobrem que o seu gráfico Tropical continua a ser a descrição mais precisa.
O True Sidereal é mais preciso que o Tropical?
Isto depende do que se entende por "preciso". O Tropical tem sido usado com sucesso durante milhares de anos na astrologia ocidental e foi o sistema que Ra Uru Hu usou para o Human Design. O True Sidereal reflete as posições astronómicas como aparecem no céu. São estruturas diferentes, cada uma com a sua própria lógica e validade. A Genetic Matrix fornece ambos para que possa comparar e determinar qual sistema descreve melhor a sua experiência.
Como é que o True Sidereal é diferente do Sideral padrão?
O Sideral padrão (como o ayanamsa Lahiri) pega no zodíaco Tropical e desloca-o por um desvio fixo para realinhar com as estrelas, mas mantém todos os doze signos a exatamente 30 graus cada. O True Sidereal vai mais longe - usa os tamanhos reais das constelações, que variam significativamente. Isto significa que o True Sidereal pode produzir resultados substancialmente diferentes do Sideral padrão, especialmente para posicionamentos perto dos limites dos signos.
Qual variante True Sidereal devo usar?
Não há uma resposta universal. Cada variante representa uma abordagem válida para mapear o céu visível. Recomendamos gerar o gráfico com as três e comparar os resultados. Muitos praticantes que trabalham com True Sidereal recomendam o ayanamsa Chimenti pela sua consistência com referências tradicionais de constelações, enquanto Kanatas é amplamente usado na astrologia sideral ocidental. O sistema Midpoint oferece uma alternativa astrologicamente refinada aos limites astronómicos da IAU.
Ra Uru Hu endossou o True Sidereal?
Ra Uru Hu desenvolveu o Human Design usando o zodíaco Tropical. Ele não usou cálculos Siderais ou True Sidereal. A Genetic Matrix oferece True Sidereal como uma ferramenta de exploração adicional para aqueles que desejam ver o seu gráfico através desta perspetiva. Fornecemos ferramentas, não doutrina - e não afirmamos saber o que Ra teria ou não teria endossado.
